Demo showing a custom number input that accepts decimals in the users native locale, but always returns a numeric format internally.

Please change the locale in the app.module.ts to experiment.

The input will only accept decimals ',' when using French.

(Example is modified from the original angular material custom form control guide, so it's got lots of things related to phone numbers still https://material.angular.io/guide/creating-a-custom-form-field-control#trying-it-out)}
